== Description ==

Plugin adds widget and shortcode with random animated gif from http://gifs4u.com to your site.

This plugin was tested with Wordpress 3.6.1.

== Installation ==

1. Upload the entire 'random-animated-gif' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now You can use widget or shortcode [animated-gif].
